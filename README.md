## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Benshi7/twinbank
   ```


2. Install dependencies:
   ```bash
   cd twinbank-backend
   pip install -r requirements.txt
   ```
   
   ```bash
   cd twinbank-spa
   npm install
   ```

3. Set-up:
   Back-end:
      ```bash
      cd twinbank-backend
      py manage.py runserver
      ```

   Front-end:
      ```bash
      cd twinbank-spa
      node run dev
      ```
